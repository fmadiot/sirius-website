4' Tutorial

This tutorial shows the first steps to create a first graphical modeling tool with Sirius.

It relies on a simple DSL and a sample model based on this DSL.

With this video, you will see how to:

  • create a Sirius project (Viewpoint Specification project)
  • initialize a Diagram to graphically vizualize and edit the model elements
  • define Nodes and Edges to display model elements on the diagram
  • add a Creation Tool to create new model elements from a palette

The step-by-step scenario of this video can be found here: https://wiki.eclipse.org/Sirius/Tutorials/4MinTutorial
