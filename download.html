Download

Sirius source code is currently being migrated to Eclipse.

Until the first official release (0.9), a development version of Sirius is already available (updated nightly).

Update Site

Copy this URL: http://download.eclipse.org/sirius/updates/nightly/0.9 to your Eclipse Installation to reach this Solution's Update Site.

Sirius is available as update site. To install via the update site URL above, copy and paste it into the "Help > Install new software" dialog.
